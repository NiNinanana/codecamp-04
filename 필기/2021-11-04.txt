반복문 for
일정 로직을 계속 반복해서 실행하는 문법

for(let i=0; i < 5; i++){

}

let i=0; : 최초식
i < 5; : 조건식
i++ : 증감식 (i++은 i = i+1과 같다)

무한루프를 조심할 것

오늘의 커리큘럼


서버와 데이터를 주고 받는 2가지 방식
동기와 비동기

동기
게시물 내용을 보내면 백엔드에서 저장을 하고 응답을 줌

게시물 등록 api 를 먼저 요청
게시물 등록이 완료 된 후
게시글 불러오기 api

하나하나 완료가 되어 응답이 모두 끝난 후 다음것을 요청

비동기
비동기는 등록이 끝나기도 전에 불러와줘라고 한다. 근데 비동기가 default이기에 동기로 바꿔줘야한다.
게시물 등록 api를 요청하고
게시물 등록이 완료 되기 전에 
게시글 불러오기 api를 요청했기에 오류 발생

비동기는 오래걸릴 때ㅅ 사용하게 된다.(이메일 전송: 3s / 게임 다운 받으면서 카톡)
한번에 모두 진행

비동기를 동기로 바꿔주는 명령어
async(에이신크) / await

사용하고 싶은거 앞에다가 넣어주면 된다

자바스크립트는 동기지만 외부에 작동하는것이 디폴트가 비동기다

비동기
async function 함수이름(){
    요청문
    실행문 // promise 언제줄진 모르지만 줄거야
}
요청과 실행이 동시에

동기
async function 함수이름(){
    await 요청문
    실행문 // 실행됨
}
요청이 끝난 후 실행

화살표함수 사용 시
const zzz = async () => {

}


비동기는 default이기에 async는 안써줘도 된다
동기로 할 때는 써줘야한다

개발자도구 network에서 http상태코드를 통해서 잘 처리되고 있는지 알 수 있다.
